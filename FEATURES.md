# Функциональность приложения Zvon

## Реализованные функции

### ✅ Аутентификация
- Регистрация новых пользователей
- Вход в систему
- JWT токены для безопасности
- Защищенные маршруты

### ✅ Серверы (Guilds)
- Создание серверов
- Просмотр списка серверов
- Управление серверами (только владелец)
- Присоединение к серверам
- Удаление серверов

### ✅ Каналы
- Создание текстовых каналов
- Создание голосовых каналов
- Создание категорий каналов
- Управление каналами
- Удаление каналов

### ✅ Сообщения
- Отправка сообщений в реальном времени
- Просмотр истории сообщений
- Редактирование своих сообщений
- Удаление сообщений (автор или владелец сервера)
- Индикатор набора текста (typing indicator)
- Форматирование даты и времени

### ✅ Пользователи
- Профили пользователей
- Статусы (online, offline, away, busy)
- Аватары
- Список участников сервера

### ✅ Real-time коммуникация
- WebSocket соединения через Socket.io
- Мгновенная доставка сообщений
- Индикатор набора текста
- Автоматическое переподключение

### ✅ UI/UX
- Современный интерфейс в стиле Discord
- Адаптивный дизайн
- Темная тема
- Интуитивная навигация
